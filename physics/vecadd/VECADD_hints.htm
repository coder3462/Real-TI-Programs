Is this program useful?

The VECADD program for the TI-83 or TI-84+ prompts you to enter two 2-D vectors. It then computes their lengths and the length and angle of their sum, and draws a picture of the two original vectors plus the sum vector. Full instructions for using the program are on the Web.

This page gives three methods to get the program onto your calculator. (The same VECADD will work on all TI-83 and TI-84+ models, including the ones with color screens.)

Method 1: File Transfer from Your Computer

If you have a TI-84+, it has a USB port. Use the free TI Connect CE software from Texas Instruments to transfer programs to your calculator.

If you have a TI-83, it has a round I/O port. Use the free TI Connect software from Texas Instruments to transfer programs to your calculator. (Careful! You want TI Connect, not TI Connect CE.)

Method 2: File Transfer from Another Calculator

If you don’t have the TI software, or your calculator can’t connect to your computer, see if a friend or classmate already has the program and can transfer it from her calculator to yours. You can transfer:

  • Between any models of TI-84+.
  • Between any models of TI-83.
  • Between any TI-83s and black-and-white TI-84+s, but not between TI-83 or TI-83+ and color TI-84+s.

Use this procedure:

  1. On the calculator that already has the program, press [2nd x,T,θ,n makes LINK] [3] and select the program VECADD.
  2. On the calculator that needs the program, press [2nd x,T,θ,n makes LINK] [►] [ENTER].
  3. On the calculator that already has the program, press [►] [ENTER].
  4. If you get a prompt about a duplicate program, choose Overwrite.

Method 3: Key in the Program

If you don’t have the TI software and can’t transfer the program from someone else’s TI, you can key in the program yourself. The complete source code is shown in the accompanying file VECADD.pdf, in the same ZIP file that you downloaded as <http://BrownMath.com/ti83/prog/vecadd.zip>.

To type in a program, begin with [PRGM] [◄] [ENTER], type in the program name VECADD, and press [ENTER]. The Programming chapter of your TI-83 or TI-84+ manual has full information about entering programs.

If you do have to type in the entire program, here are a couple of hints:

  • For the double-quote mark, press [ALPHA + makes "].
  • : is [ALPHA] [.].
  • → is the STO→ key.
  • Xmin, Ymax, ΔX, and similar are on the [VARS] [1] menu.
  • R►Pr and R►Pθ are in the [2nd APPS makes ANGLE] menu. (On the classic TI-83, the menu is [2nd MATRX makes ANGLE].)
  • If you can’t find a particular function in your calculator’s menus, you can always get to it with [2nd 0 makes CATALOG].
